##: Làm sao để tìm file adb.exe 
B1: Trên android Studio -> chọn file -> settings -> languages & Framework -> Android SDk
B2: Nhìn cái link thư mục Android SDK Location: ........
B3: Đi theo đường link thư mục để tìm -> platform-tools 
B4: Kéo file adb.exe vào terminal (nếu không được thì làm B5) và nhập pair + địa chỉ hướng dẫn connect qua mã ghép nối
B5: chuột phải vào file adb.exe -> copy as path rồi làm tương tự phía trên
